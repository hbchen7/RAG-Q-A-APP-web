# 项目名 AIChats

# 技术栈

## 前端

- Vue3--前端框架

### 代码规范

Eslint9 + Prettier + Husky + lint-staged

### 其他

- pnpm --包管理工具
- ElementPlus --UI组件库
- ElementPlus/icons --图标库
- axios--http请求库
- pinia--状态管理库
- vite--打包工具
- sass--css预处理器
